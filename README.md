# Teilstückberechnung

Dieses Projekt enthält ein kleines GUI-Tool zur Berechnung von Teilstücken. Es wurde in Python programmiert und bietet eine grafische Oberfläche, mit der Berechnungen interaktiv durchgeführt werden können.

## Funktionen  
- Eingabe von Start- und Endwert (oder Länge/Ausrichtung je nach Anwendung)  
- automatische Berechnung des Teilstücks  
- einfache grafische Bedienung (Fenster, Buttons, Eingabefelder)  
- Ausgabe des Ergebnisses direkt im Fenster  

## Voraussetzungen  
- Python 3.x  
- evtl. zusätzliche Bibliotheken: z. B. `tkinter` (Teil der Standard-Bibliothek)  
- Für die GUI: eventuell weitere Module (falls verwendet)
